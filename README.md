# アプリ名
SideViewAction

# 概要
2Dの横スクロール型のゲームです。
ファミコンのスーパーマリオブラザーズ的なものです。
←↑↓→でキャラクターを操作することができます
spaceキーでジャンプができます。

# 制作背景
ゲームを実際に作成してみたいと思い
